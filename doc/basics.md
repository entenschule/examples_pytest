## installation

``` 
tilman@t570:~/learn_py/examples_pytest sudo apt install python3.8-venv
tilman@t570:~/learn_py/examples_pytest python3 -m venv env
tilman@t570:~/learn_py/examples_pytest source env/bin/activate
(env) tilman@t570:~/learn_py/examples_pytest python --version
Python 3.8.10

(env) tilman@t570:~/learn_py/examples_pytest pip install pytest
(env) tilman@t570:~/learn_py/examples_pytest pytest --version
pytest 7.4.0
```
